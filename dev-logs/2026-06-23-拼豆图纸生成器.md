# 拼豆图纸生成器

- **日期**: 2026-06-23
- **状态**: 已实现 + 优化完成，待验证

## 同日优化

### 1. 高清图纸导出

- 导出分辨率从 36px/格 提升至 **144px/格（4x）**
- 所有元素等比放大：网格线、刻度字体、色卡块
- 色卡区域高度动态计算，适配颜色数量

### 2. 调色板颜色全部对齐 MARD 291

- 4 套预设调色板 48 种颜色，全部替换为最近 MARD 291 色号
- 使用 `findClosestMardCode()` 按 RGB 欧几里得距离就近匹配
- 颜色名称保持不变，仅 HEX 值更新

### 3. 色块 Hover 显示 MARD 色号

- `_buildSwatches()` 中为每个色块添加 `title` 属性
- 格式：`草莓奶昔 · F14`
- 精确匹配优先，无匹配则就近查找

## 背景

用户需要根据当前拼豆图案生成带色号的「拼豆图纸」，方便对照制作实物拼豆。

参考文件：`图纸生成参考.html`（使用 html2canvas CDN）

## 实现

### 入口

「📷 导出 ▼」菜单新增「📐 生成图纸」选项

### 图纸规格

| 要素 | 实现 |
|------|------|
| 网格主体 | HTML `<table>`，每格显示 MARD 色号 + HEX 底色 |
| 四边刻度 | 上 1→N / 下 N→1 / 左 1→M / 右 M→1 |
| 颜色统计 | 底部色卡栏：色块预览 + 色号 + 数量 `×N` |
| 色号映射 | HEX → MARD（反向查表），非 MARD 色就近匹配 |
| 导出 PNG | Canvas 绘制整张图纸，`{作品名}_图纸.png` |
| 样式 | 纯白背景、浅灰网格线、Arial 字体 |

### 技术要点

- **零依赖**：不用 html2canvas，自行 Canvas 绘制导出
- **反向映射** `MARD_HEX_TO_CODE`：字面量初始化，O(1) 查表
- **邻近色匹配** `findClosestMardCode()`：RGB 欧几里得距离，用于自定义颜色

### 模态框结构

```
┌─ 📐 拼豆图纸 ──────────────────┐
│   1  2  3 ... N  (列刻度)       │
│ 1 [A5][C8]...         N (右)   │
│ 2 ...                          │
│ M ...                          │
│   N ... 2  1  (列刻度)         │
│ ┌──┐ ┌──┐                      │
│ │A5│ │C8│ ...  (色卡统计)     │
│ ×5  ×3                         │
│ [📷 导出图纸PNG] [关闭]        │
└────────────────────────────────┘
```
