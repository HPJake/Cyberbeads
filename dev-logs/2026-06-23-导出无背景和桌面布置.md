# 2026-06-23 导出无背景 + 桌面布置功能

## 功能一：导出无背景图案

### 需求
- 点击导出选项 → 弹出预览窗口
- 默认勾选「包含背景」
- 取消勾选 → 导出透明背景 PNG（纯透明，无白底无底板）

### 实现
- **`ExportManager.exportToCanvas(grid, mode, realisticEdges, transparent)`**：提取渲染逻辑到独立方法，`transparent` 参数控制白底
- **`ExportManager.export()`**：重构为调用 `exportToCanvas` + `toBlob` 下载
- **导出预览模态框**（`#export-preview-modal`）：checkerboard 背景预览透明效果、checkbox 实时切换、确认下载/取消
- **App 方法**：`_showExportPreview()`、`_renderExportPreview()`、`_confirmExportDownload()`、`_closeExportPreview()`
- 修改 `_initUI()` 中 `.export-opt` 点击处理，改为调用 `_showExportPreview(mode)`

## 功能二：桌面布置

### 需求
- 右下角「🎨 布置桌面」按钮
- 上传 `.json` 拼豆存档 或 `.png` 图片
- JSON 默认「扁平效果（普通）」渲染；PNG 直接显示
- 随机位置（避开菜单栏/调色栏/标题区）
- 拖拽移动、滚轮/双指缩放（0.5x ~ 3x）
- 图层在桌面参考图之下（z-index 上限 19）
- 右键菜单 / 触摸长按 500ms
- localStorage 持久化（key: `cyber-beads-desk-items`）

### 架构
- **新类 `DeskArrangementManager`**（~270 行）：
  - `items[]` 数组管理所有图案 `{ id, type, dataUrl, saveData?, renderMode?, x, y, scaleX, scaleY, zIndex }`
  - `_renderJSONToDataURL(data, mode)`：构建临时网格 → 用 CELL=40 渲染全尺寸 canvas → 缩放到 ≤400px → 输出 data URL
  - 三种渲染模式：bead-mode（3D圆柱）、flat-normal（普通扁平）、flat-realistic（逼真边缘）
  - `_handleJSONFile()` / `_handlePNGFile()`：文件读取分发
  - `_createDOM()`：创建 `.desk-canvas-item` 容器 + 绑定 pointer/wheel/touch 事件
  - `_bringToTop()` / `_bringToFront()` / `_sendToBack()` / `_removeItem()`：图层管理
  - `_showMenu()` / `_hideMenu()` / `_handleMenuAction()`：右键菜单
  - `_loadFromStorage()` / `_saveToStorage()`：localStorage 持久化
- **右键菜单**：
  - JSON：豆子效果 / 扁平效果（普通）/ 扁平效果（逼真） / 置最上层 / 置于最下层 / 移除
  - PNG：置最上层 / 置于最下层 / 移除
- **触摸长按**：pointerdown 启动 500ms 定时器，移动 >3px 或 pointerup 取消
- **拖拽图层**：pointerdown → `_bringToTop` + `.dragging` 类临时 `z-index: 9999`；pointerup → 恢复 `item.zIndex`（上限 19）
- **初始尺寸**：max 200px 宽度，通过 `transform: scale()` 控制（修复了双重缩放 bug）
- 在 `App` 构造函数中初始化：`this.deskArranger = new DeskArrangementManager(this)`

### 体积控制
- JSON 渲染后缩放到 max 400px，data URL 约 20-50KB（64×64 网格）
- PNG 直接用原始 data URL

### 文件变更
- `cyber-beads.html`：单文件修改
- CSS：新增 ~80 行（导出预览 + 桌面布置全部样式）
- HTML：新增 ~50 行（导出预览模态框 + 桌面布置按钮/图层/菜单/文件输入）
- JS：新增 ~350 行（exportToCanvas 重构 + DeskArrangementManager 类 + App 导出预览方法）
