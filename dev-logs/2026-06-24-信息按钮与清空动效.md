# 2026-06-24 信息按钮与清空动效

## 需求

1. 左上角 "i" 圆形按钮，点击弹出操作说明模态框
2. 清空动效：确认清空后豆子自由落体跌落，熨烫豆子整块掉落

## 改动

### `index.html`

- **信息按钮**：`<button id="info-btn">` 加在 `<body>` 层级，静音按钮之前，左上角固定
- **帮助模态框**：`#help-modal`，含基本操作/熨烫/导出/布置桌面四栏说明
- **清空动画画布**：`<canvas id="clear-anim-canvas">` 覆盖层，动画时显示

### `css/style.css`

- **`#info-btn`** — 圆形按钮，桌面 `top: 14px; left: 20px; 36×36`，手机 `top: 6px; left: 6px; 28×28`
- **`#clear-anim-canvas`** — `position: fixed; inset: 0; z-index: 150; pointer-events: none;`
- 手机端 `#info-btn` 缩小适配

### `js/app.js`

- **信息按钮逻辑**：点击弹出/关闭 `#help-modal`，点击遮罩层也可关闭
- **清空动画** `_playClearAnimation()`（2026-06-24 重写）：
  - 通过 `getBoundingClientRect` 精确定位豆子在视口中的位置
  - **熨烫连通块识别**：BFS 四方向搜索，将相邻熨烫豆子（`ironed === true` 或 `heat >= 0.92`）归为一个连通块
  - 多豆连通块作为整体掉落（共享 `dy` + `vy`），保持内部相对位置不变
  - 单颗熨烫豆子和未熨烫豆子各自独立掉落
  - **先画 overlay 第一帧 → 清 grid → 立即 `drawAllBeads()` 强制重绘主画布**（解决时序问题：主画布与 overlay 无缝衔接）
  - 物理：纯垂直自由落体，重力加速度 **1200 px/s²**，无水平偏移、无旋转
  - 正常豆子：`ellipse` + `darkenColor` 内壁（与 Renderer 一致）
  - 熨烫豆子：调用 `drawIronedFlatCell`（与 Renderer 一致）
  - 帧间隔钳制 `dt ≤ 0.1s`，防止 death spiral

### `js/sound-engine.js`

- **`playSpill()`** — 洒落音效：白噪声 + 低通滤波频率从 3000Hz 降到 200Hz，持续 0.4s

## 效果

- 清空时未熨烫豆子各自垂直下落，熨烫块整块保持形状下落
- 重力 1200px/s²，加速度明显，约 0.6~0.9s 全部出屏
- overlay 与主画布同步切换，视觉上豆子从原位直接落下
- 动画期间禁止操作（overlay z-index: 150 拦截所有指针事件）
