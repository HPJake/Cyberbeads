# 逼真模式：从圆角方案迁移到逐边圆弧波浪花边算法

- **日期**: 2026-06-22
- **状态**: 已实现，待验证

## 背景

用户反馈：真实的拼豆熨烫后，外边缘应该呈现「波浪扇贝花边」，而非圆角矩形。这是因为豆子是圆形，熨烫后无邻居的方向保持圆弧边缘，有邻居的方向因相互挤压变为直线。

## 新算法

### 物理模型

- 每颗豆子是圆形（直径 ≈ 格子大小）
- 熨烫后，豆子摊平填满方格
- 某方向有邻居 → 互相挤压 → 边界为直线
- 某方向无邻居 → 保持圆形弧边 → 半圆弧凸出格子
- 多个自由边在角落自然交汇 → 波浪扇贝花边

### 逐边路径构建（顺时针）

```
对于豆子方格 (x, y, size, size) 的四条边：

上边 (x,y)→(x+size,y):  有上邻居 → lineTo | 无上邻居 → arc 向上凸出
右边 (x+size,y)→(x+size,y+size): 有右邻居 → lineTo | 无右邻居 → arc 向右凸出
下边 (x+size,y+size)→(x,y+size): 有下邻居 → lineTo | 无下邻居 → arc 向下凸出
左边 (x,y+size)→(x,y):  有左邻居 → lineTo | 无左邻居 → arc 向左凸出
```

### 圆弧计算 (`drawEdgeArc`)

- 弧线连接边的两个端点，向外凸出 `bulge` 像素
- 圆心在边的垂直平分线上，`D = (bulge² - S²/4) / (2*bulge)`
- 半径 `R = sqrt(S²/4 + D²)`
- `bulge < 0.5` 时退化为直线

### 过渡动画

- `flatness = clamp((heat - 0.92) / 0.08, 0, 1)`
- `bulge = EDGE_BULGE * flatness`
- heat=0.92 → bulge=0 (无花边，纯方形)
- heat=1.0 → bulge=18 (完整花边)

## 代码变更

### 新增

- `CONFIG.EDGE_BULGE: 6` — 最大凸出像素（调小至约 15% 格子大小，接近真实拼豆）
- `drawEdgeArc(ctx, x1, y1, x2, y2, bulge, ox, oy)` — 逐边圆弧辅助函数

### 重写

- `drawIronedFlatCell` — 逼真路径从四角圆角矩形改为逐边圆弧算法
  - 移除圆角/溢出/jitter 逻辑
  - 保留 neighbor 检测、flatness 过渡、纹理叠加
  - 保留 try-finally 安全保护

### 移除

- `drawRoundRectArcTo` — 旧的圆角矩形路径函数，不再使用
