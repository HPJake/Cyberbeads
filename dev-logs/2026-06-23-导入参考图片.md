# 导入参考图片功能

- **日期**: 2026-06-23
- **状态**: 已实现，待验证

## 需求

玩家导入参考图片 → 方形裁切 → 同时生效两个效果：
1. **描画层**：画布下方半透明参考（可调透明度），用于描画
2. **桌面参考窗**：浮动画框参考图，可拖动缩放

## 实现

### UI 布局

```
🖼 (导入按钮，调色栏左侧独立框)

┌──┐  ┌──────────────────────┐
│☑描│  │    bg-canvas         │
│画  │  │    ref-canvas ← 半透明│
│滑  │  │    bead-canvas       │
│块  │  │    fx-canvas         │
│50%│  └──────────────────────┘
└──┘

┌──────┐
│ 画框 │  ← 浮动桌面参考窗（可拖动、可调大小）
│ 参考 │
└──┘
```

### HTML 新增

| 元素 | 说明 |
|------|------|
| `#ref-import-box` + `#btn-import-ref` | 导入按钮（调色栏左侧） |
| `#ref-file-input` | 隐藏文件输入 |
| `#canvas-area` | 画布区域 flex 行容器 |
| `#ref-side-controls` | 左侧描画控件（开关+滑块+数值） |
| `#ref-canvas` | 参考图层 canvas（bg 与 bead 之间） |
| `#desktop-ref-window` | 浮动桌面参考窗（含画框 + 关闭按钮） |
| `#crop-modal` | 裁切弹窗（含 canvas + 网格叠加层） |

### CSS 新增

- `#ref-import-box` — 与调色栏风格一致的独立框
- `#canvas-area` — flex 行布局，侧控件 + 画布
- `#ref-side-controls` — 垂直排列控件，圆角卡片风格
- `#ref-opacity-slider` — `writing-mode: vertical-lr` 竖排滑块
- `#ref-canvas` — `pointer-events: none` 透传点击
- `#desktop-ref-window` — 固定定位，可拖动；`#desktop-ref-frame` 画框效果
- `#desktop-ref-close` — 圆形关闭按钮
- `#crop-canvas-wrap` — 深色背景裁切区
- `#crop-overlay` — 方形虚线框 + 阴影遮罩

### JS 方法（App 类）

| 方法 | 说明 |
|------|------|
| `_initReferenceSystem()` | 初始化：绑定按钮、文件输入、滑块、开关、桌面窗拖拽 |
| `_importReference()` | 触发文件选择 |
| `_showCropModal(file)` | 读取文件 → Image → 打开裁切 UI |
| `_buildCropUI(img)` | Canvas 绘制图片 + 方形裁切框 + N×N 网格线 + 缩放/平移 |
| `_applyReference(dataUrl)` | 存储裁切结果 → 启用侧控件 → 绘制描画层 → 显示桌面窗 |
| `_drawReferenceUnderlay()` | 在 ref-canvas 上绘制半透明参考图 |
| `_showDesktopRef(dataUrl)` | 显示浮动桌面参考窗（默认位置：画布左侧） |
| `_hideDesktopRef()` | 隐藏桌面参考窗 |
| `_initDesktopRefDrag()` | 拖动/缩放逻辑 + 边界约束 |
| `_constrainDesktopRef()` | 限制桌面窗不超出视口、不覆盖工具栏/调色栏 |

### 交互细节

**裁切弹窗**：
- 滚轮缩放（以鼠标位置为中心）
- 拖拽平移
- 方形裁切框（360px）+ N×N 网格线叠加
- 框外半透明遮罩
- 双指触摸缩放/平移

**描画层**：
- 默认 50% 透明度
- 竖排滑块调节（0~100%）
- 勾选框开关
- 缩放/调整网格后自动重绘

**桌面参考窗**：
- 画框装饰（双层边框 + 阴影）
- 拖动移动
- 右下角拉动调整大小
- 不可覆盖工具栏/调色栏（自动碰壁）

### 修改的现有代码

- `Renderer.resize()` — 新增 ref-canvas 尺寸同步
- `App` 构造函数 / resize-confirm / continue-modal / import-restore — 新增 `_drawReferenceUnderlay()` 调用
