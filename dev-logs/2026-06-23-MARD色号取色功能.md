# MARD 色号自定义取色功能

- **日期**: 2026-06-23
- **状态**: 已实现，待验证

## 背景

用户希望通过 MARD 字母数字色号（如 A5、C8）快速查找到对应颜色并选取。

## 实现

### MARD 291 映射表

- 来源: `https://pindou.skin/colors.html` → `colors.js`
- 用 Node.js 解析 JS 对象，提取 MARD 291 色号的 name→hex 映射
- 共 291 色，15 个系列（A-R, T, Y, ZG）
- 嵌入为 `MARD_COLORS` 常量（~4.2KB）

### UI 变更

在取色面板中，预览框与「应用」按钮之间增加：

- `#mard-code-input` — 文本输入框 + 内嵌「确认」按钮（右侧）
- `#mard-confirm-btn` — 确认按钮，位于输入框内部右侧

布局：`[MARD 色号标签] → [ 输入框... | 确认 ] → [应用]`

### 交互逻辑

1. 点击「🎨 自选色」打开取色面板
2. 输入 MARD 色号（如 `A5`），点击「确认」或按 Enter
3. 查找 `MARD_COLORS[code]`：
   - **找到** → 取色器色相/饱和度/明度跳到对应位置 + 预览框显示颜色（不应用、不关闭面板）
   - **未找到** → 输入框边框变红闪烁 600ms 提示
4. 用户确认颜色无误后 → 点击「应用」→ 颜色选中并加入自定义调色板
