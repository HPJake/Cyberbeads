# 2026-06-24 图纸文字自适应 + 熨烫提速

## 一、图纸色号文字颜色自适应

### 问题
图纸格子和色卡中色号文字统一为深色（`#222`），暗色背景上完全看不清。

### 修复
新增 `contrastTextColor(hex)` 辅助函数，基于相对亮度（0.299R + 0.587G + 0.114B）判断：
- 亮度 > 0.5 → 深色文字 `#1a1a1a`
- 亮度 ≤ 0.5 → 白色文字 `#ffffff`

涉及位置：
- 图纸预览 HTML 表格格子 `td.style.color`
- 色卡 `.color-block` `block.style.color`
- PNG 导出格子 `ctx.fillStyle`
- PNG 导出色卡 `ctx.fillStyle`

## 二、熨烫提速

三项措施叠加，总体效率提升约 4-5 倍：

### 1. 提高加热速率
- `HEAT_RATE`: 0.025 → **0.045** (1.8x)
- `HEAT_ADJACENT`: 0.01 → **0.025** (2.5x)
- 周边格子加热几乎与中心持平，扫过时整体加热更均匀

### 2. 补帧插值
- `InputManager._ironAt()`: 在两次 `pointermove` 之间用直线插值填充跨越的格子
- 快速移动指针时不再遗漏中间格子，每条路径都被完整覆盖

### 3. 已有基础：3×3 刷子
- `heatCell()` 原本就加热中心格 + 8 邻居 = 3×3 区域，无需额外扩大
