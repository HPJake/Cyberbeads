# 2026-06-24 参考图分辨率 + 桌面装饰移除 + 调整画布保留豆子

## 一、参考图导入分辨率提升

### 问题
裁切确认时从低分辨率显示 canvas（固定 360×360px）提取，导致参考图画质严重压缩。

### 修复
`_buildCropUI` 中确认裁切逻辑改为 **直接从原始高清 Image 对象提取**：
- 根据当前 `panX / panY / zoom` 计算裁切区域在原图中的坐标 `(imgX, imgY, imgW)`
- 输出分辨率 = `min(2400, max(360, nativePx))`，自适应于原图在该区域的像素密度
- 用户放大 → 裁切分辨率接近原图；缩小看全图 → 最高 2400px 上限
- 超出原图边界的区域填充 `#222` 深色背景
