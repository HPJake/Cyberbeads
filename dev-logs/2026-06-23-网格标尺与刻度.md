# 2026-06-23 网格标尺与刻度 + 滑块调整 + 阴影修复 + 引导弹窗

## 一、网格默认值与滑块调整

### 变更
- `DEFAULT_GRID`: 16 → **15**
- `MIN_GRID`: 8 → **5**
- `MAX_GRID`: 64 → **100**
- 滑块: `min="5" max="100" value="15" step="5"`
- JS 吸附: `Math.round(v / 5) * 5`

## 二、每 5 格标线

在底板 `drawPegboard()` 中，每隔 5 行/5 列绘制较粗的参考线 (rgba(160,140,120,0.28), lineWidth=1)，便于数格子对位。

## 三、数字刻度

### 右侧控件
- 新增 `#ruler-side-controls` 面板，位于画布右侧，与左侧参考图控件对称
- CSS 样式与 `#ref-side-controls` 一致
- 包含「刻度」勾选框 (`#ruler-toggle`)

### 刻度渲染
- 勾选后，`Renderer.showRulers = true`，在画布四边显示数字刻度
- 刻度编号方式与蓝图导出一致：
  - **上边**: 1 → N（左到右）
  - **下边**: N → 1（右到左，镜像）
  - **左边**: 1 → N（上到下）
  - **右边**: N → 1（下到上，镜像）
- 每个格子都有数字标签（非仅5的倍数）
- 字体: 10px Arial, 颜色 #8B7355

### 画布间距
- 开启刻度后，`resize()` 使用 `pad = RULER_PAD (22px)`（默认 8px），为刻度数字腾出空间
- `_positionSideControls()` 扩展为同时定位左/右面板，右侧面板定位在 `visualRight + gap`

### 刻度状态
- 不持久化，刷新后恢复默认（关闭）
- 切换时自动重设画布尺寸和重绘

## 四、去除桌面图案阴影

删除了 `.desk-canvas-item::after` 伪元素，消除了上传图案周围的阴影边框。

## 五、布置桌面引导弹窗

点击「🎨 布置桌面」不再直接弹出文件选择框，先显示引导弹窗：
- 说明支持格式（.json / .png）和操作方式（拖拽/缩放/右键菜单）
- 「📁 上传文件」按钮触发文件选择并关闭弹窗
- 支持点击遮罩关闭
