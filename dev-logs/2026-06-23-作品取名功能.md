# 作品取名功能 + 色号确认按钮优化

- **日期**: 2026-06-23
- **状态**: 已实现，待验证

## 色号确认按钮优化

- 确认按钮从输入框外侧移入输入框内部右侧
- 按钮高度调整：`height:auto; font-size:10px; padding:0 6px` 不再超出输入框
- 确认行为改为**仅同步取色器**（更新色相/饱和度/明度 + 预览），不应用颜色、不关闭面板
- 用户确认预览无误后，点击「应用」才最终选中颜色

## 作品取名功能

### UI（优化后）

- 位置：调色栏下方、画布上方
- 样式：**中式门牌框** — 双层边框（外层浅木色、内层深木色四角收边）、米纸渐变底色、细阴影
- 结构：`#artwork-name-frame` 包裹 `#artwork-name-input`
- 交互：点击输入文字，placeholder "给作品取个名字..."，最大 40 字符
- **失焦自动隐藏光标**：点击画布或页面其他位置，输入框自动失焦

### 导出

- 有作品名 → 导出 `{作品名}.png`
- 无作品名 → 回退为 `cyber-beads-{mode}-{timestamp}.png`

### 暂不实现

- localStorage 存档（后续再考虑）

## 代码变更

- HTML: 新增 `#artwork-name-bar` 区块 + `#artwork-name-frame` 门牌框
- CSS: 中式门牌双层边框样式（`::before`/`::after` 四角收边）
- JS: `ExportManager.export` 文件名逻辑 + document click 失焦
