# 2026-06-25 图纸预览自适应与自动裁切

## 需求

1. **预览图纸自适应**：大画布时预览窗口只能看到部分图纸需要滚动条 → 等比缩放让玩家一眼看到完整图纸
2. **图纸自动裁切**：勾选后自动检测豆子包围盒，四周留2格边距，裁掉空白区域

## 实现

### `index.html`

- `.blueprint-options` 新增 `✂️ 自动裁切（四周留2格边距）` 勾选框 `#bp-crop`

### `js/export-manager.js`

- 新增 `ExportManager.getBeadBoundingBox(grid)`：遍历所有豆子找 min/max 行列，四周+2格边距，clamp 到网格边界。无豆子时返回 null
- `exportToCanvas` 新增可选参数 `cropBox`（图纸裁切复用了此方法）

### `js/app.js`

**`_showBlueprintModal`**：
- `buildData()` 按 crop 状态动态构建 gridData，传入 `getBeadBoundingBox()` 结果
- `renderPreview()` 每次调用 `buildData()` + `_renderBlueprintPreview()` + `requestAnimationFrame(() => _fitBlueprintPreview())`
- crop 复选框 change 事件触发 `renderPreview()`
- 导出按钮也调用 `buildData()` 获取最新 crop 参数传给 `_exportBlueprint()`

**`_renderBlueprintPreview`**：
- 新增参数 `r1, c1`（裁切后左上角在原网格中的偏移）
- 列标尺显示 `colStart + i` ~ `colEnd - i`（原网格坐标）
- 行标尺显示 `rowStart + i` ~ `rowEnd - i`（原网格坐标）

**`_fitBlueprintPreview`**（新增）：
- 重置 paper transform → 测量自然尺寸
- 计算可用空间（modal 宽 - 48px, max-height 650px）
- 若 `scale < 1`：`transform: scale()` + `transform-origin: top center` + 容器高度截断 + overflow hidden
- 若 `scale >= 1`：恢复默认 overflow

**`_exportBlueprint`**：
- 新增参数 `r1, c1`
- PNG 导出标尺使用原网格坐标 `colStart/End`、`rowStart/End`

## 行为

- 默认不勾选 → 原有行为（完整网格图纸）
- 勾选「自动裁切」→ 自动检测豆子最小包围盒 + 四周2格边距 → 表格即时重渲染 → 标尺显示正确行/列号 → 导出 PNG 同样裁切
- 画布无豆子时勾选 → 跳过裁切，保持原尺寸
- 图纸过大时自动等比缩放至预览窗口内，无需滚动条

## Bug 修复 (2026-06-25)

### Bug 1: 勾选「自动裁切」后图纸无法渲染

- **根因**: `getBeadBoundingBox()` 调用 `grid.getAllBeads()`，但 `GridModel` 没有此方法 → JS 异常 → `buildData()` 崩溃
- **修复**: 改用 `grid.occupiedCells`（Set 集合，"r,c" 键值对）直接遍历

### Bug 2: 豆子贴左边缘时裁切后左边无2格边距

- **根因**: `Math.max(0, r1-2)` / `Math.max(0, c1-2)` 在豆子位于第0列时将边距 clamp 掉了
- **修复**: 移除 clamp，允许 crop 区域超出画布边界（虚拟空行/列）。`getBead()` 对越界坐标返回 null，渲染时空格子自然呈现

### Bug 3: 预览界面网格白底与色卡/文字分裂

- **根因**: `_fitBlueprintPreview()` 用 `setTimeout(50)` + `getBoundingClientRect` 测量纸片尺寸。`setTimeout` 不能保证 DOM 布局已完成，测出的高度可能不含色卡/文字区域 → scale 偏大 → `preview.style.height` 偏大 → `overflow: hidden` 把纸片底部截断 → 白底不完整
- **修复**: 改为纯数据驱动——根据 gridData 计算色卡行数 × 行高、文字行数 × 行高，加上标尺/内边距，精确算出纸片总高度。同步调用无时序问题。加 10px 安全余量确保不截断。
