# 图纸镜像与署名功能

- **日期**: 2026-06-23
- **状态**: 已实现，待验证

## 需求

在拼豆图纸预览界面新增三个功能：

1. **镜像勾选框** — 豆子图案左右翻转，刻度保持不变
2. **显示作品名称勾选框** — 图纸下方显示黑色作品名称（取自门牌框输入）
3. **填写作者大名勾选框** — 勾选后弹出输入框，图纸下方显示灰色 `by {作者名}`

三项变更后预览即时重绘，导出 PNG 同步更新。

## 实现

### HTML 变更

图纸模态框 `#blueprint-modal` 内新增：

```html
<!-- 勾选框行 -->
<div class="blueprint-options">
  <label><input type="checkbox" id="bp-mirror"> 镜像</label>
  <label><input type="checkbox" id="bp-show-name"> 显示作品名称</label>
  <label><input type="checkbox" id="bp-show-author"> 填写作者大名</label>
</div>

<!-- 作者名输入行（默认隐藏） -->
<div id="bp-author-row" style="display:none;">
  <input type="text" id="bp-author-input" placeholder="请输入作者大名..." maxlength="20">
  <button class="btn primary" id="bp-author-confirm-btn">确定</button>
</div>

<!-- 图纸下方文字显示区 -->
<div id="blueprint-text-area" style="display:none;">
  <div id="bp-artwork-name-text">作品名（黑色）</div>
  <div id="bp-author-name-text">by 作者（灰色）</div>
</div>
```

### CSS 新增

- `.blueprint-options` — 勾选框行 flex 居中布局
- `#bp-author-row` / `#bp-author-input` — 作者输入行样式，与 MARD 色号输入框风格一致
- `#blueprint-text-area` — 文字显示区，中文字体栈

### JS 重构

#### `_showBlueprintModal()` 拆分

- 原方法包含一次性的全部渲染逻辑
- 重构为：构建 gridData → 初始化 checkbox 状态 → 抽取 `renderPreview()` 闭包
- `renderPreview()` 调用新方法 `_renderBlueprintPreview(gridData, rows, cols, opts)`
- 三个 checkbox 的 `onchange` 均绑定 `renderPreview`
- 「填写作者大名」checkbox 额外控制 `#bp-author-row` 显隐
- 作者「确定」按钮和 Enter 键触发重绘

#### 新增 `_renderBlueprintPreview(gridData, rows, cols, opts)`

- `opts.mirror` → 渲染 grid 时列索引反转 `c = mirror ? cols - 1 - ci : ci`
- 刻度渲染不变（四边数字保持原方向）
- `opts.showName && artName` → 显示作品名称
- `opts.showAuthor && authorName` → 显示 `by {作者}`
- 两个勾选框独立控制，互不依赖

#### `_exportBlueprint()` 增强

- 新增 `opts` 参数（mirror, showName, showAuthor, authorName）
- 镜像：`srcC = mirror ? cols - 1 - c : c`，视觉位置不变，源数据列翻转
- 文本区域动态计算：`TEXT_LINE_H = 28*S`，`TEXT_PAD = 12*S`
- Canvas 高度 = 原有高度 + TEXT_AREA
- 作品名：黑色 bold 16*S px
- 作者名：灰色 (#999) 14*S px，前缀 `by `

## 交互流程

```
打开图纸 → 三个勾选框默认未勾
  ├─ 勾选「镜像」→ 预览即时左右翻转
  ├─ 勾选「显示作品名称」→ 若已填写作品名，预览下方出现黑色名称
  ├─ 勾选「填写作者大名」→ 展开输入框 → 输入名字 → 确定/回车 → 预览出现灰色 by 行
  └─ 点击「导出图纸PNG」→ 导出包含当前所有选项的最终效果
```

## 技术要点

- 镜像不修改原始 gridData，仅在渲染时反转列索引
- 刻度永远不镜像（用户确认：只镜像图案）
- `getOpts()` 闭包在导出时实时读取 checkbox 状态
- 两个文字勾选框独立，可以只显示作者不显示作品名
