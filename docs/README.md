# Cyber Beads (赛博拼豆)

虚拟熔珠拼豆模拟器 — 单文件 HTML5 网页游戏。

## 玩法

1. 在调色板中选择颜色
2. 点击画布格子放置豆子
3. 使用镊子工具移除豆子
4. 点击"熨烫"进入加热模式，在豆子上移动鼠标使其融化融合
5. 可选开启"逼真模式"获得波浪边缘融合效果
6. 点击"完成定形"锁定画布
7. 导出 PNG 图片

## 功能

- **多套调色板**：多种精选配色方案
- **逼真边缘**：熨烫后豆子边缘自然波浪融合
- **撤销/重做**：最多 60 步历史记录
- **自动保存**：通过 localStorage 持久化保存
- **画布缩放**：支持滚轮缩放和拖拽平移
- **导出**：支持豆子效果（带孔）和熨烫效果（扁平）两种 PNG 导出

## 技术栈

纯前端，零依赖。HTML + CSS + JavaScript 单文件（~2900 行）。

## 浏览器兼容

Chrome / Edge / Firefox 最新版。
