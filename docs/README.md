# Cyber Beads (赛博拼豆)

虚拟熔珠拼豆模拟器 — 纯前端 HTML5 网页应用。

## 玩法

1. 在调色板中选择颜色
2. 点击/拖拽画布格子放置豆子
3. 使用镊子工具移除豆子
4. 点击"熨烫"进入加热模式，在豆子上移动鼠标使其融化融合
5. 可选开启"逼真模式"获得波浪边缘融合效果
6. 点击"完成定形"锁定画布
7. 导出 PNG 图片或拼豆图纸

## 功能

- **多套调色板**：马卡龙、复古、原色、大地等多套预设 + 自定义取色
- **MARD 色号系统**：支持 MARD 291 标准色号，自定义颜色自动就近匹配
- **逼真边缘**：熨烫后豆子边缘呈波浪扇贝花边，模拟真实熨烫挤压效果
- **撤销/重做**：最多 60 步历史记录，支持 Ctrl+Z / Ctrl+Y
- **自动保存**：通过 localStorage 持久化保存
- **画布缩放**：支持滚轮缩放和拖拽平移
- **参考图**：导入图片作为临摹参考，支持透明度调节；桌面端另有浮动参考窗
- **布置桌面**：将作品或 PNG 图片布置在页面背景中，支持拖拽/缩放/图层管理/渲染切换
- **手机适配**：响应式布局，支持全屏模式，触屏优化

## 导出

| 模式 | 说明 |
|------|------|
| ⭕ 豆子效果 | 带孔圆柱形豆子，模拟未熨烫状态 |
| 🟫 熨烫效果（普通/逼真） | 扁平融合状态，逼真模式带扇贝花边边缘 |
| 📐 拼豆图纸 | 4x 高清 PNG，格状图纸 + MARD 色号标注 + 四边标尺 + 色号图例 |

### 拼豆图纸详解

图纸专为打印对照制作而设计，包含：
- **网格色块** — 每格标注 MARD 色号，文字颜色自动适配背景对比度
- **四边标尺** — 上下列号 + 左右行号，双向对称标注
- **色号图例** — 底部自动罗列所用颜色，显示色块 + 色号 + 用量
- **可选信息** — 作品名 / 作者名
- **底部水印** — 淡灰色 `- Cyberbeads 赛博拼豆 -`

**自动裁切**：勾选后自动裁剪四周空白行列，保留 2 格边距。标尺编号跟随原网格坐标。

**镜像模式**：左右翻转输出，适配熨烫后背面阅读。
