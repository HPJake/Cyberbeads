# 🦾 赛博拼豆 Cyber Beads

**线上拼豆、模拟、拼豆图纸制作工具** — 在浏览器里自由创作熔珠拼豆图案，无需实体材料。

🎮 **试玩链接**：[hpjake.github.io/Cyberbeads](https://hpjake.github.io/Cyberbeads/)

---

## 🎯 游戏规则

拼豆（fuse beads）是一种手工创作：将彩色小珠子按图案摆在钉板上，用熨斗加热使珠子融化粘合，冷却后形成完整图案。

本作完整模拟了这一过程：

1. **选色** — 在底部调色板选择豆子颜色
2. **放置** — 点击/拖拽在网格上放置豆子
3. **熨烫** — 点击"🔥 熨烫"进入加热模式，在豆子上移动鼠标/手指模拟熨斗加热，豆子逐渐融化变平
4. **定形** — 点击"✅ 完成定形"锁定作品
5. **导出** — 导出 PNG 高清图片或拼豆图纸

## 🖱 操作逻辑

### 基础操作

| 操作 | 方式 |
|------|------|
| 放置豆子 | 点击 / 拖拽画布格子 |
| 擦除豆子 | 切换到镊子工具（屏幕底部工具卡片），点击豆子 |
| 撤销 / 重做 | 工具栏按钮 或 `Ctrl+Z` / `Ctrl+Y` |
| 缩放画布 | 滚轮 / 双指捏合 |
| 平移画布 | 拖拽空白区域 |

### 颜色系统

- **预设调色板**：马卡龙、复古、原色、大地等多套配色方案，点击切换
- **自定义颜色**：点击"🎨 自选色"打开取色器，支持屏幕取色（Chrome/Edge）
- **MARD 色号**：支持 MARD 291 标准色号输入，自动匹配最接近的颜色

### 熨烫系统

- 进入熨烫模式后，在豆子上**长按/拖拽**模拟加热
- 温度条显示当前加热程度，豆子从圆柱形逐渐变扁平
- **✨ 逼真模式**：开启后熨烫边缘呈波浪扇贝花边，模拟真实熨烫挤压效果
- 可撤销熨烫步骤，也可退出熨烫继续编辑

### 参考图

- 导入图片作为临摹参考，叠加在画布上
- 可调节透明度，支持裁切
- 桌面端另有**浮动参考窗**，可自由拖拽缩放

### 导出模式

| 模式 | 效果 |
|------|------|
| ⭕ 豆子效果 | 带孔圆柱形豆子，模拟未熨烫状态 |
| 🟫 熨烫效果 | 扁平融合状态，可选逼真边缘 |
| 📐 拼豆图纸 | 格状图纸 + 色号标注，可打印对照制作 |

### 存档

- 自动保存到浏览器（localStorage），刷新不丢失
- 支持导出/导入 `.json` 存档文件
- 可为作品命名

### 布置桌面

- 将拼豆作品或 PNG 图片布置在页面背景中
- 支持拖拽、缩放、图层管理
- 拼豆存档可在三种渲染效果间切换

## 📱 手机端与电脑端的区别

| | 💻 电脑端 | 📱 手机端 |
|------|-----------|-----------|
| **工具栏按钮** | 完整图标 + 文字 | 紧凑模式：部分按钮仅图标 |
| **存档按钮** | 💾 存档 ▼ | 💾（仅图标，点击弹出菜单） |
| **参考图按钮** | ⛰️ 参考图（在调色栏内） | 参考图（纯文字，调色栏内） |
| **布置桌面按钮** | 🎨 布置桌面 | 🎨 布置 |
| **全屏按钮** | 隐藏 | 右上角固定显示，点击进入全屏 |
| **网格设置** | 工具栏内直接调整 | 点击 ⚙ 展开面板 |
| **调色板** | 色块自动换行 | 色块缩小，紧凑换行 |
| **浮动参考窗** | ✅ 支持拖拽/缩放 | ❌ 不含（仅叠加参考） |
| **工具抽屉** | 较大卡片 | 缩小为紧凑尺寸 |
| **标题** | 顶部居中 | 左下角小字 |
| **缩放指示器** | 显示 | 显示 |
| **快捷键** | Ctrl+Z/Y 全部可用 | 触屏操作（外接键盘可用） |
